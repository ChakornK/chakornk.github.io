This project was inspired by [Project Sekai Stickers](https://github.com/TheOriginalAyaka/sekai-stickers), which allows you to create stickers with characters from Project Sekai and your own custom text. This project allows you to do the same with characters from Genshin Impact. The stickers can then be downloaded as PNG files and used on your favorite social media platforms.

![Character selector](/projects/genshin-sticker-maker/2.png)

## Tech stack

- **Frontend**: Preact, Tailwind, Konva
